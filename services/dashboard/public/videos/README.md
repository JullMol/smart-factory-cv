# Sample Videos for Testing

Place factory/construction CCTV videos here for testing PPE detection.

## Recommended Video Sources

Search YouTube for:
- "Industrial CCTV footage"
- "Construction site safety video"
- "Factory floor surveillance"
- "PPE detection test video"

## Video Requirements

- **Format:** MP4 (H.264)
- **Resolution:** 720p or 1080p
- **Content:** Workers with and without PPE (hardhat, safety vest)
- **Angle:** Top-down or angled view (like real CCTV)

## File Naming

Name your video files descriptively:
```
factory_entrance.mp4
assembly_line.mp4
loading_dock.mp4
machine_shop.mp4
```

## Usage

The dashboard will automatically detect and use videos from this folder.
If no videos are found, sample videos from Google are used as fallback.
